## Getting a Client
For this guide, I'm going to be using [FluffyChat](https://fluffychat.im/en/), which is available on the [Google Play Store](https://play.google.com/store/apps/details?id=chat.fluffy.fluffychat), [Apple App Store](https://apps.apple.com/us/app/fluffychat/id1551469600) and [F-Droid](https://f-droid.org/packages/chat.fluffy.fluffychat/). 
<sub>(though lets be honest, you probably wont be using f-droid)</sub>

## Signing In
Once you have fluffychat installed and open, click the "Login with Matrix ID" at the very bottom (Screen may differ between Android and iOS)
